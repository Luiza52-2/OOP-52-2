Команды один раз на устройство

Команды для регистрации новых данных

git config --global user.name "Your name"
git config --global user.email "your-email@example.com"

Команды один раз для проекта
git init
git remote add origin https://github.com/Luiza52-2/OOP-52-2.git

часто используемые команды
git status
git add .

Комманда для создания коммита
git commit -m "LS: Lesson_1 1 added learn python oop"
Команда для просмотра ветки
git branch

Комманда для отправки коммита в репозитории
git push origin master
______________________________________________
Команда для удаления директории и все что  она сожержит
git rm -r --cached .idea

Базовые команды линукса
ls писок файлов и директории
ls -a список файлов и директории которые являются скрытыми
